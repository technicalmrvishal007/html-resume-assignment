## HTML resume website Page

This project is a single-page resume website created using only HTML.
The main goal of this project is to practice basic HTML structure and semantics
without using CSS or JavaScript.

## Project Overview

- Single-page resume website
- Built using only HTML
- Clean and readable structure
- Includes all required resume sections

## Sections Included

- Header (Name and Role)
- About Me
- Skills
- Experience (Fresher)
- Projects
- Education
- Contact Details

## Technologies Used

- HTML5

## How to Run the Project

1. Clone or download this repository
2. Open the `index.html` file
3. Run it in any web browser (Chrome, Edge, Firefox, etc.)
